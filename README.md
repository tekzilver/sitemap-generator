Tekzilver Sitemap Generator

A simple, open-source Chrome Extension that generates XML sitemaps for any website.
Features

    Privacy First: We do not collect any information from users, nor do we track what you do in any way. All crawling happens locally in your browser.
    Respects robots.txt: The crawler checks and respects the website's crawling rules.
    Limit: Crawls up to 500 URLs to prevent performance issues.
    Export: Exports a clean XML Sitemap compatible with Google Search Console.

Installation Instructions

    Download the source code from GitHub (Click "Code" -> "Download ZIP").
    Extract the downloaded ZIP file to a folder on your computer.
    Open Google Chrome and go to chrome://extensions/.
    Toggle the "Developer mode" switch in the top-right corner.
    Click the "Load unpacked" button in the top-left.
    Select the folder where you extracted the files.
    The Tekzilver Sitemap Generator extension is now installed!

Usage

    Click the extension icon in your toolbar.
    Enter a website URL (e.g., https://example.com).
    Click "Generate XML Sitemap".
    Wait for the progress bar to finish.
    Save the file when prompted.
